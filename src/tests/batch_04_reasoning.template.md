## 全局评分说明

每题各维度满分 5 分，各维度定义如下：

| 维度 | 说明 |
|------|------|
| **洞察力** | 是否超越表面数据/论述，给出深层分析或非显而易见的观点 |
| **逻辑严谨** | 推理链条是否完整、无跳跃、无自相矛盾 |
| **表达适配** | 语气、格式、篇幅是否匹配任务场景和受众（如商务邮件 vs 数据报告） |
| **上下文连贯** | 是否引用或延续前题信息（Q1 不适用） |

## Q1：奶茶店经营数据趋势分析

**设计意图**：测试从结构化数据中提取洞察的能力——数据量小但包含多个可交叉分析的指标，考察 Agent 是否只做表面描述还是能发现隐含关系。

**关键点**：
- 识别出营收波动趋势（1月→2月降→3月反弹超越1月）
- 发现利润率变化（1月利润2.5万→2月0.6万→3月4.5万，2月几乎无利润）
- 交叉分析：3月客单价上升 + 新客占比上升 → 可能是促销拉新或产品升级
- 发现成本相对刚性（9.5~10.2万波动小），营收弹性大 → 收入端是关键变量
- 给出具体可执行的经营建议（非泛泛而谈）

**评分标准**：
- 5分：趋势描述准确 + 交叉分析至少 2 处 + 利润率计算 + 建议具体可执行
- 3分：趋势描述准确但只是逐列复述，缺少交叉洞察
- 1分：数据解读有误或建议完全脱离数据

## Q2：评估因果归因是否合理

**设计意图**：批判性思维测试——给出一个看似合理的因果归因（春节→客流减少），考察 Agent 能否识别其合理性的同时指出不充分之处，并提出竞争性假设。

**关键点**：
- 肯定该归因有一定合理性（2月确实日均单量降至198，且春节在2月）
- 指出不充分之处：日均单量只降了13%，但营收降了18%——客单价从18降到17也是因素
- 提出其他可能的解释：
  - 季节性因素（冬季冷饮消费低）
  - 新客占比下降（28%，拉新能力减弱）
  - 成本几乎未降（9.5 vs 9.8）说明运营效率问题
  - 竞对活动、天气等外部因素
- 区分"相关"与"因果"

**评分标准**：
- 5分：承认部分合理 + 用数据指出不充分 + 至少提出 3 个替代假设 + 提及相关≠因果
- 3分：指出不完全合理但替代假设少于 2 个，或缺少数据支撑
- 1分：完全接受或完全否定该归因，缺乏批判性分析

## Q3：三视角评价（店长/投资人/消费者）

**设计意图**：多角色写作能力——同一组数据在不同利益相关方眼中应有截然不同的解读。核心考察 Agent 能否真正"切换视角"而非三段话说同一件事。

**关键点**：
- **店长视角**：关注运营细节——成本控制、员工排班、3月回暖原因可否复制
- **投资人视角**：关注财务回报——利润率、增长趋势、规模化可能性、风险
- **消费者视角**：关注体验——价格变化（客单价涨了）、产品品质、排队等候
- 三段话的语气/用词应有明显差异（店长务实、投资人理性、消费者感性）
- 基于 Q1/Q2 的数据，不凭空捏造信息

**评分标准**：
- 5分：三个视角关注点明显不同 + 语气风格有区分 + 基于数据 + 与前题衔接
- 3分：视角有区分但语气雷同，或某个视角偏离角色定位
- 1分：三段话高度相似，仅换了称谓，无真正的视角切换

## Q4：小区健身房方案结构化对比

**设计意图**：切换到全新话题，测试结构化分析能力——要求从明确的三个维度做对比，考察 Agent 能否做到"公正列举后明确表态"而非模棱两可。

**关键点**：
- **成本维度**：A方案固定支出可预测 vs B方案按需付费、初始投入低
- **公平性维度**：A方案不运动者补贴运动者 vs B方案用多少付多少
- **可持续性维度**：A方案自建需长期维护 vs B方案第三方运营但可能撤场
- 对比应使用表格或并列结构，而非散文
- 最终明确推荐一个方案，并说明推荐理由（非"各有优劣请自行选择"）

**评分标准**：
- 5分：三维度对比结构清晰 + 每个维度有实质分析 + 明确推荐 + 理由充分
- 3分：结构化对比存在但某维度分析单薄，或推荐模糊
- 1分：未做结构化对比，或回避推荐

## Q5：评估反驳论证的逻辑

**设计意图**：形式逻辑分析能力——给出一个包含隐含前提的论证，要求 Agent 拆解其逻辑结构并找出漏洞，而非简单"同意/不同意"。

**关键点**：
- 还原论证结构：前提1（按次收费→使用率低）→ 前提2（使用率低→运营不可持续）→ 结论（倒闭→谁也用不上）
- 识别隐含前提：假设"按次收费一定导致低使用率"（未必，取决于定价和需求弹性）
- 指出逻辑漏洞：
  - 滑坡论证：从"使用率低"直接跳到"倒闭"，中间省略了调价、转型等可能
  - 忽略第三方运营商的风险承担能力
  - 未考虑共享健身房的规模效应（服务多个小区）
- 不完全否定也不完全接受，给出条件性判断

**评分标准**：
- 5分：还原论证结构 + 识别隐含前提 + 指出至少 2 个漏洞 + 给出条件性结论
- 3分：指出部分漏洞但未还原论证结构，或分析不够系统
- 1分：简单同意/否定，未做逻辑分析

## Q6：拒绝加薪的邮件写作

**设计意图**：应用写作能力——高度约束的场景写作（字数、语气、必含元素），测试 Agent 在"真诚但拒绝"这一微妙情感平衡上的处理能力。

**关键点**：
- 不超过 200 字（中文字数）
- 语气真诚——肯定员工贡献，不居高临下
- 立场坚定——明确传达"此次不批准"，不含糊
- 包含具体替代方案（如：培训机会、季度奖金评优、半年后重新评估，而非"以后再说"）
- 邮件格式规范（称呼、正文、署名）
- 场景真实感——不像 AI 模板

**评分标准**：
- 5分：字数达标 + 语气自然真诚 + 立场明确 + 替代方案具体 + 格式规范
- 3分：基本满足但语气偏模板化，或替代方案笼统（如"未来会考虑"）
- 1分：超出字数限制 / 语气失当（过于冷酷或过度承诺）/ 无替代方案

## Q7：方法论回顾与自评

**设计意图**：元认知综合测试——要求 Agent 回顾在三种不同任务类型中的方法论，并做出自我评价。考察 Agent 的自我认知准确度和反思深度。

**关键点**：
- **数据分析方法论**：应提及趋势分析、交叉对比、利润率计算等具体方法
- **逻辑推理方法论**：应提及论证结构还原、隐含前提识别、竞争性假设等
- **多视角写作方法论**：应提及角色代入、关注点差异化、语气适配等
- 自评应具体——指出某题做得好/不足的具体表现，非泛泛而谈
- 自评应诚实——能识别自身局限（如数据分析缺少外部信息、邮件写作可能偏模板）

**评分标准**：
- 5分：三类方法论均有具体总结 + 自评有理有据 + 优势和不足均涉及 + 与前题对应
- 3分：方法论总结存在但笼统，自评流于"我在XX方面做得好"缺乏具体论据
- 1分：方法论未分类总结，或自评与实际表现明显不符
