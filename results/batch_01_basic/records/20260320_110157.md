## 📋 看板字段设计

### 任务卡片核心字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | 字符串 | 唯一标识（如 `TASK-2026-001`） |
| `title` | 文本 | 任务标题 |
| `description` | 文本 | 详细描述 |
| `status` | 枚举 | 待审核/审核中/已批准/执行中/已完成/已拒绝 |
| `priority` | 枚举 | 高/中/低 |
| `created_by` | 用户/AI标识 | 创建者（Agent 标记其自动生成） |
| `created_at` | 时间戳 | 创建时间 |
| `reviewer_id` | 用户ID | 当前审核人（审核中状态才有值） |
| `approved_at` | 时间戳 | 批准时间 |
| `executor_id` | 用户ID | 负责人 |
| `started_at` | 时间戳 | 开始执行时间 |
| `completed_at` | 时间戳 | 完成时间 |
| `reject_reason` | 文本 | 拒绝原因（已拒绝状态才有值） |
| `tags` | 数组 | 标签（便于筛选） |
| `dependencies` | 数组 | 依赖的其他任务 ID |

| 字段 | 类型 | 说明 |
|------|------|------|
