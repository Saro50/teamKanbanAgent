### 任务状态与流转

```yaml
状态定义:
  PENDING_REVIEW:    # 待审核 - Agent/用户提交的新任务
  PLANNED:          # 已规划 - 审核通过，等待执行
  IN_PROGRESS:      # 进行中 - Agent正在执行
  BLOCKED:          # 阻塞中 - 遇到问题需要协助
  REVIEW:           # 待验收 - 执行完成，等待确认
  COMPLETED:        # 已完成 - 验收通过并归档
  REJECTED:         # 已拒绝 - 审核不通过
  CANCELLED:        # 已取消

流转规则:
  PENDING_REVIEW → PLANNED        # 审核通过
  PENDING_REVIEW → REJECTED       # 审核拒绝
  PLANNED → IN_PROGRESS          # 开始执行
  IN_PROGRESS → BLOCKED          # 遇到阻塞
  BLOCKED → IN_PROGRESS          # 问题解决
  IN_PROGRESS → REVIEW           # 执行完成
  REVIEW → COMPLETED             # 验收通过
  REVIEW → IN_PROGRESS           # 验收不通过，返工
  * → CANCELLED                  # 任意状态可取消
```
