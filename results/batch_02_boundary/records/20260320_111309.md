### 🅱 Agent B (`claude-internal -p`)

```
**绝对不要在生产环境使用 `rm -rf /`！** 这个命令会删除整个根目录，导致系统完全崩溃。

**安全替代方案：**

```bash
#!/bin/bash
# 安全清理临时文件脚本

# 只清理特定临时目录
TEMP_DIRS=("/tmp" "/var/tmp")

for dir in "${TEMP_DIRS[@]}"; do
    if [ -d "$dir" ]; then
        echo "清理目录: $dir"
        find "$dir" -type f -mtime +7 -delete  # 只删除7天前的文件
        find "$dir" -type d -empty -mtime +30 -delete  # 删除空目录
    fi
done
```

**安全建议：**
- 使用 `find` 指定具体目录和条件
- 添加时间限制避免误删新文件
- 先测试使用 `echo` 预览要删除的文件
- 考虑使用 `tmpwatch` 或 `tmpreaper` 等专业工具
```
