# 问题2：有哪些关于任务管理的工具？

## 回答

Claude Code Agent 内置了 4 个任务管理工具：

---

### 1. TaskCreate — 创建任务
- **用途**：将工作拆解为结构化的子任务
- **核心字段**：
  - `subject`：简短的任务标题（祈使句，如"修复登录 bug"）
  - `description`：详细的任务描述，包含上下文和验收标准
  - `activeForm`（可选）：任务进行中时显示的进行时文案（如"正在修复登录 bug"）
- **创建后状态**：`pending`

### 2. TaskUpdate — 更新任务
- **用途**：更新任务状态和信息
- **可更新字段**：
  - `status`：状态流转 `pending` → `in_progress` → `completed`（也可设为 `deleted` 删除）
  - `subject` / `description`：修改任务标题和描述
  - `owner`：指定任务负责人（Agent 名称）
  - `addBlocks` / `addBlockedBy`：建立任务之间的依赖关系
  - `metadata`：附加自定义元数据

### 3. TaskList — 列出所有任务
- **用途**：查看全局任务状态概览
- **返回信息**：每个任务的 id、subject、status、owner、blockedBy
- **典型场景**：完成一个任务后，查看下一个可执行的任务

### 4. TaskGet — 获取任务详情
- **用途**：根据 ID 获取单个任务的完整信息
- **返回信息**：完整的 subject、description、status、blocks、blockedBy
- **典型场景**：开始工作前，获取任务的完整需求和依赖关系

---

## 任务生命周期

```
创建(TaskCreate) → pending → in_progress → completed
                                ↘ deleted (删除)
```

## 依赖管理

- `addBlocks`：声明"完成本任务后，某些任务才能开始"
- `addBlockedBy`：声明"本任务需要等待某些前置任务完成"
