# 问题1：你有哪些能力？

## 回答

作为 Claude Code Agent，我具备以下核心能力：

### 1. 代码开发能力
- **读取文件**：使用 Read 工具读取任意文件内容（代码、配置、图片、PDF、Jupyter Notebook 等）
- **编写与编辑文件**：使用 Write/Edit 工具创建新文件或精确修改现有文件
- **执行命令**：通过 Bash 工具运行终端命令（构建、测试、部署等）
- **代码搜索**：通过 Glob（文件名匹配）和 Grep（内容搜索）快速定位代码

### 2. 任务管理能力
- **创建任务**（TaskCreate）：将复杂工作拆解为结构化的子任务
- **更新任务**（TaskUpdate）：跟踪任务状态（pending → in_progress → completed）
- **查询任务**（TaskList / TaskGet）：查看全局进度和任务详情
- **任务依赖**：建立任务之间的阻塞/被阻塞关系

### 3. 多 Agent 协作能力
- **启动子 Agent**（Agent 工具）：将子任务分发给专门的 Agent 并行执行
  - `general-purpose`：通用任务处理
  - `Explore`：快速代码库探索
  - `Plan`：架构设计与实施规划
  - `claude-code-guide`：Claude Code 使用指南

### 4. 规划与决策能力
- **进入规划模式**（EnterPlanMode）：对复杂实现进行方案设计，获取用户审批后再执行
- **交互式提问**（AskUserQuestion）：在执行过程中向用户澄清需求

### 5. 信息检索能力
- **Web 搜索**（WebSearch）：获取最新信息
- **Web 内容获取**（WebFetch）：获取并分析网页内容
- **GitHub 操作**：通过 gh CLI 进行 PR、Issue 等 GitHub 工作流操作

### 6. Git 版本控制能力
- 提交代码（commit）、创建分支、创建 PR
- 支持 Worktree 隔离工作

### 7. Jupyter Notebook 编辑能力
- 通过 NotebookEdit 工具对 `.ipynb` 文件进行单元格级别的编辑
